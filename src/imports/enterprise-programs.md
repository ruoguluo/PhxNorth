PAGE 1
Global Talent Mobility Program
(previously Strategic Talent Campaigns)

Hero Title:
Global Talent Mobility Program

Hero Description:
A structured talent mobility infrastructure enabling individuals to participate in enterprise hiring campaigns across regions, industries, and roles.

Subheading:
Accessible to all registered users of the platform.

SECTION 1 — How to Participate

Step Flow:

1. Activate Talent Mobility
Users enable the Talent Mobility function in their dashboard.

2. Set Role Preferences
Users configure their focus areas including:
• Country / Region
• Market
• Industry
• Professional domain
• Role title or seniority level

3. Receive Opportunities
The system pushes relevant enterprise campaigns in real time.

4. Participate
Users can either:
• Recommend a candidate
• Self-nominate for the opportunity

SECTION 2 — 5D Intelligent Matching

Explain platform capability:

PhxNorth does not use simple binary matching.
Instead, the platform evaluates alignment between:

• Candidate 5D profile
• Enterprise role 5D requirements

Dimensions include:
• Hard skills
• Soft skills
• Decision logic
• Risk calibration
• Network capital

The system highlights potential alignment considerations.

Example:
A candidate may match strongly in expertise but the system may indicate that decision style or risk calibration could become a focus during enterprise interviews.

This guidance helps users make higher-quality referrals.

SECTION 3 — Process Governance

Once a referral or self-nomination is submitted:

• Users complete structured information required by the platform.
• AI system evaluates the submission.
• The campaign follows a professional executive search workflow.
• Progress updates appear in the user’s dashboard.

SECTION 4 — User Benefits

• Participate in global talent mobility opportunities
• Access curated enterprise hiring campaigns
• Receive intelligent matching insights before recommending
• Track referral progress transparently
• Build professional reputation within the ecosystem

Primary CTA:
Activate Talent Mobility

PAGE 2
Commercial Consultation Projects
(previously Enterprise Advisory Projects)

Hero Title:
Commercial Consultation Projects

Hero Description:
Structured consulting engagements where enterprises collaborate with experienced professionals through platform-managed advisory projects.

SECTION 1 — How to Participate

Step Flow:

1. Activate Commercial Consultation
Users enable the function in their dashboard.

2. Set Project Interests
Users select focus areas including:
• Industry
• Project type
• Expertise domain

3. Discover Opportunities
Users receive relevant consultation project signals or can browse available opportunities.

4. Apply for Project
Users submit an application including:
• Self-introduction related to project needs
• Relevant project experience
• Successful case studies
• Time commitment availability

SECTION 2 — Project Selection Process

After submission:

• AI evaluates the applicant's 5D compatibility with project requirements.
• The platform project manager reviews applications.
• Shortlisted consultants are invited to a one-on-one online interview.
• Final consultants are confirmed and onboarded into the project.

SECTION 3 — Project Governance

Once selected:

Projects are managed jointly by:

• AI system monitoring progress
• Platform project manager coordinating execution

This ensures structured delivery, milestone tracking, and professional governance.

SECTION 4 — User Benefits

• Participate in real enterprise consulting projects
• Work with structured project governance
• Increase professional exposure and credibility
• Build verified consulting track records
• Access cross-industry strategic engagements

Primary CTA:
Activate Commercial Consultation
