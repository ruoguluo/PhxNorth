----------------------------------------

Profile

My Mentorship

   1. Instant Mentorship
      - Large Online / Offline toggle switch (very visible)
      - Set Instant Rate (£ per 30min)
      - Set Available Time Windows
      - Max Concurrent Sessions

      If Online:
         - Show Waiting Requests
         - Accept / Decline buttons
         - Countdown timer

      If Offline:
         - Show message: "You are currently offline"
         - Button: Go Online

   2. Scheduled Mentorship
      - Upcoming Sessions
      - Calendar View
      - Reschedule / Cancel
      - Pre-session Notes

   3. My Mentees
      - Active Long-term Mentees
      - Past Mentees

      Each mentee card shows:
         - Name
         - Industry
         - # Sessions completed
         - 5D Compatibility Score
         - Button: Continue Mentorship
         - Option: Remove from Long-term

      After each session, mentor can click:
         "Save as Long-term Mentee"

----------------------------------------

My Workshops

   1. PhxNorth Workshops
      - Mentor can set workshop matching conditions:
            Industry
            Topic Keywords
            Seniority Level
            Format (Panel / Deep-dive / Strategy)

      When a new workshop matches mentor conditions:
            Show badge: "New Matching Topic Available"
            Show workshop card with:
                Topic
                Target audience
                Date
                Button: Apply as Speaker

   2. My Workshops
      - Create Workshop Topic
      - Manage Registrations
      - View Attendance
      - View Feedback
      - Earnings Overview

      If no workshop created:
            Show invitation page:
            "Raise your own workshop topic and build authority."
            CTA: Create Your First Workshop

----------------------------------------

My Courses

If mentor has created courses:
      - My Classes
      - My Students
      - Cohort Analytics
      - Completion Rate

      Show invitation page:
      "Turn your mentorship framework into a structured course."
      CTA: Develop a Course
